# Hostel Leave Management System

A web-based system to manage hostel leave requests, attendance, and student monitoring.

---

## Features

- User Authentication (Student / Warden)
- Leave Application System
- Leave Approval / Rejection
- Email Notifications (PHPMailer)
- Reports Dashboard
- Attendance System
- Unauthorized Student Detection

---

## Modules

### Student
- Login
- Apply Leave
- View Leave Status
- View Attendance

### Warden
- Approve / Reject Leave
- Add Students
- Mark Attendance
- View Unauthorized Students
- View Reports

---

## Technologies Used

- Frontend: HTML, CSS
- Backend: PHP
- Database: MySQL
- Server: XAMPP
- Email: PHPMailer

---

## Key Concepts

- Role-based authentication
- Session management
- Prepared statements (SQL Injection prevention)
- Password hashing
- Attendance + Leave integration

---

## 📊 Database Tables

- users

- hostel_leaves
- leave_types
